# Luna Client

Advanced Minecraft utility client for educational purposes - Version 2.0.0

## Features

### Combat Modules
- AutoCrystal - Automated crystal combat system
- KillAura - Enhanced combat targeting
- AutoTrap - Automatic trap placement
- Surround - Self-protection system
- And many more combat utilities

### Movement Modules
- Speed - Multiple speed modes
- Fly - Flight capabilities
- ElytraFly - Enhanced elytra control
- Step - Automatic step-up
- And more movement enhancements

### Render Modules
- ESP - Entity highlighting
- HoleESP - Safe hole detection
- NameTags - Enhanced player tags
- PlaceRender - Block placement visualization

### Client Features
- Custom GUI system
- Module configuration
- Keybind management
- Friend system

## Building

```bash
./gradlew build
```
